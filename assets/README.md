# Assets Folder

This folder exists for supporting files that are not core Python source code.

At the moment, it is not a major part of the chatbot submission.

## Intended Use

If the project grows, this folder would be a good place for things like:

- screenshots
- small demo media
- diagrams
- documentation images

## Current Importance

Right now, you can treat this folder as optional support space rather than a key part of the chatbot logic.

The main implementation lives in:

- `src/`
- `faqs.json`
- `tests/`
